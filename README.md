# HR Management System

Unified platform for PTO Tracking and Staff Evaluations

## Architecture

- **Frontend**: React + TypeScript + Material-UI
- **Backend**: Google Apps Script
- **Database**: Google Sheets
- **Auth**: Google Workspace

## Modules

1. **PTO Tracker** - Manage time-off requests, balances, and approvals
2. **Staff Evaluations** - 360-degree performance reviews and competency management
